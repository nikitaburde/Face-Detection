# Face Detection using OpenCV 

This is a simple **Face Detection** project using **OpenCV** in Python. The script detects faces in an image or video stream (webcam) and draws rectangles around them.

## 📌 Features

✅ Detects faces in an image\
✅ Real-time face detection using a webcam\
✅ Uses OpenCV's **Haar Cascade Classifier**

## 🛠️ Installation

1️⃣ **Install OpenCV**

```sh
pip install opencv-python
```

2️⃣ **Clone the Repository**

```sh
git clone https://github.com/nikitburde/Face-Detection.git
cd Face Detection
```

---

## 🚀 How to Run

### 🔹 Face Detection from an Image

1. Place an image (`people.jpg`) in the project folder.
2. Run the script:
   ```sh
   python face_detection.py
   ```

### 🔹 Real-time Face Detection (Webcam)

Run:

```sh
python face_detection_video.py
```

Press **'q'** to exit.
